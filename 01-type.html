TYPE

TYPE es una propuesta de identidad visual para un evento cultural de tipografía orientado a diseñadores y estudiantes interesados con un rango de edad de entre los 20 y 40 años.

El sistema gráfico combina una paleta vibrante, composiciones dinámicas y recursos modulares que resaltan el carácter expresivo de las letras. El proyecto incluyó el desarrollo de piezas editoriales y promocionales, como afiches, cronogramas y aplicaciones de marca (postales, tote bags, etc.), buscando transmitir una experiencia visual contemporánea y participativa.
